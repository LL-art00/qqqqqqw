ai大师共享版网站：互联网分享方案

方案 A：永久公网网站

使用文件：
ai-master-share-site-deploy.zip

把这个 zip 上传到支持 Node.js 的云平台或服务器。

必须设置环境变量：
AI_MASTER_PASSWORD=你的访问密码
OPENAI_API_BASE=https://macode.cloud/v1
OPENAI_API_KEY=你的 API Key
AI_MODEL=gpt-5.5
FALLBACK_MODELS=gpt-5.5
COOKIE_SECURE=true

启动命令：
npm start

平台会给你一个公网网址，例如：
https://你的项目名.onrender.com

方案 B：临时公网链接

适合临时发给别人试用。

先安装 cloudflared，然后运行：
启动临时公网分享-cloudflared.bat

命令窗口会出现类似这样的地址：
https://xxxx.trycloudflare.com

把这个地址和访问密码发给别人即可。

注意：
1. 临时链接依赖你的电脑开机和脚本持续运行。
2. 永久公网网站更适合长期分享。
3. 别人使用会消耗你的 API Key 额度。
4. 不要上传 config.local.json。
